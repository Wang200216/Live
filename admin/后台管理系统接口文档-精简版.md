# 后台管理系统接口文档 - 精简版

## 📘 文档概述

本文档定义了后台管理系统的核心接口，实现以下功能：
1. **查看小程序数据**：用户、投票、AI内容等
2. **控制直播**：开始/停止直播、调整投票数据
3. **控制AI识别**：启动/停止AI智能识别

---

## 🎯 统一响应格式

### 成功响应
```json
{
  "success": true,
  "data": { ... },
  "message": "操作成功",
  "timestamp": 1698765432000
}
```

### 错误响应
```json
{
  "success": false,
  "message": "错误描述",
  "timestamp": 1698765432000
}
```

## 一、数据查询接口（查看小程序数据）

**功能**：获取小程序实时数据，包括用户数、直播状态、投票统计等

## 二、直播控制接口

### 2.1 开始直播

**接口**：`POST /api/admin/live/start`

**功能**：启动直播，小程序通过WebSocket接收推送并自动播放

**请求**：
```json

  "streamId": "stream_uuid_123",     // 直播流ID（必填）
  "autoStartAI": true,               // 是否自动启动AI识别
  "notifyUsers": true                // 是否推送通知给用户
}
```

**响应**：
```json
{

  "data": {
    "liveId": "live_uuid_789",
```

## 四、WebSocket实时推送

### 4.1 连接地址

```
ws://192.168.31.189:8000/ws
```

### 4.2 消息类型

| 消息类型 | 说明 |
|---------|------|
| `state` | 初始状态同步 |
| `live-started` | 直播已开始 |
| `live-stopped` | 直播已停止 |
| `votes-updated` | 票数已更新 |
| `ai-started` | AI识别已启动 |
| `ai-stopped` | AI识别已停止 |
| `ai-status-changed` | AI状态已变更 |
| `ai-content-added` | AI内容已添加 |
| `ai-content-deleted` | AI内容已删除 |

### 4.3 连接流程

1. **客户端连接WebSocket**
2. **服务器发送初始状态**（包含当前直播状态、票数、AI状态等）
3. **后台管理系统操作时，服务器推送实时更新到小程序**

## 七、小程序端改动

小程序需要添加WebSocket连接和消息处理：

```javascript
// 1. 连接WebSocket
connectWebSocket() {
  const ws = uni.connectSocket({
    url: 'ws://192.168.31.189:8000/ws'
  });
  
  ws.onMessage((res) => {
    const message = JSON.parse(res.data);
    this.handleWebSocketMessage(message);
  });
}

// 2. 处理WebSocket消息
handleWebSocketMessage(message) {
  switch (message.type) {
    case 'live-started':
      this.isLiveStarted = true;
      this.liveStreamUrl = message.data.streamUrl;
      break;
    case 'votes-updated':
      this.topLeftVotes = message.data.leftVotes;
      this.topRightVotes = message.data.rightVotes;
      break;
    case 'ai-started':
      this.isListening = true;
      break;
    // ... 其他消息处理
  }
}
```
