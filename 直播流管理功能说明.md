## 📖 管理员使用流程

### 场景:配置腾讯云直播

#### 步骤1: 访问后台

```
http://192.168.31.189:8000/admin
```

#### 步骤2: 进入直播流管理

点击左侧导航 **"📡 直播流管理"**

#### 步骤3: 添加直播流

1. 点击 **"➕ 添加直播流"** 按钮
2. 填写表单:
   ```
   流名称: 腾讯云主播间
   流地址: https://play.yourdomain.com/live/room1.m3u8
   流类型: HLS (.m3u8) - 推荐
   流描述: 腾讯云直播，1080P高清
   是否启用: ✓
   ```
3. 点击 **"保存"**

#### 步骤4: 使用直播流

1. 点击左侧 **"🎬 直播控制"**
2. 在下拉框选择 "腾讯云主播间"
3. 点击 **"🚀 开始直播"**
4. 小程序立即开始播放 ✅

#### 步骤5: 编辑或删除 (可选)

**编辑**:
- 在列表中点击 **"✏️ 编辑"**
- 修改信息后保存

**删除**:
- 点击 **"🗑️ 删除"**
- 确认删除 (正在使用的流无法删除)

## 📊 数据结构

### Stream对象

```typescript
interface Stream {
  id: string;           // 唯一标识 (自动生成)
  name: string;         // 流名称
  url: string;          // 流地址
  type: 'hls' | 'rtmp' | 'flv'; // 流类型
  description?: string; // 流描述 (可选)
  enabled: boolean;     // 是否启用
  createdAt: string;    // 创建时间 (ISO 8601)
  updatedAt: string;    // 更新时间 (ISO 8601)
}
```

### 存储示例 (streams.json)

```json
[
  {
    "id": "stream-1730419500000-abc123def",
    "name": "腾讯云主播间",
    "url": "https://play.yourdomain.com/live/room1.m3u8",
    "type": "hls",
    "description": "腾讯云直播，1080P高清",
    "enabled": true,
    "createdAt": "2025-10-31T08:10:00.000Z",
    "updatedAt": "2025-10-31T08:10:00.000Z"
  },
  {
    "id": "stream-test",
    "name": "测试流",
```
